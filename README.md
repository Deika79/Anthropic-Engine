# 🌌 Anthropic Engine

> Explore alternative universes by modifying the fundamental constants of physics.

🔗 Live Demo: https://anthropic-engine.vercel.app/

---

## 🧠 Concept

**Anthropic Engine** is an interactive scientific simulation platform that explores how variations in fundamental physical constants affect the emergence of complex structures and potentially life-supporting universes.

Inspired by the **anthropic principle**, this project allows users to navigate a parameter space of possible universes and visualize their viability in real time.

---

## ⚙️ What it does

Users can modify:

- Fine-structure constant (α)
- Strong nuclear force strength
- Electron mass
- Cosmological constant

And observe how these impact:

- ⭐ Stellar stability
- ⚛️ Formation of heavy elements
- 🧪 Chemical complexity
- 🌍 Habitability potential

---

## 🔥 Key Features

### 🌈 Continuous Habitability Modeling
- Replaced hard constraints with **soft physical penalties**
- Produces smooth gradients instead of binary outcomes
- Enables realistic exploration of parameter space

### 🗺️ Heatmap Visualization
- Real-time rendering of habitability across parameter space
- Nearest-neighbor interpolation over simulation grid
- Color-coded viability (red → green)

### 🎯 Interactive Parameter Exploration
- Dynamic sliders controlling fundamental constants
- Immediate feedback on universe viability
- Visual marker for current universe

### 🧬 Coupled Physical Models
System composed of modular scientific models:

- Stellar formation model
- Nucleosynthesis model
- Chemistry viability model
- Habitability aggregation model

These interact to produce **non-trivial emergent regions** in parameter space.

## 🧪 Simulation Engine

The simulation pipeline:

1. Evaluate stellar stability
2. Evaluate heavy element formation
3. Compute chemistry score
4. Aggregate into habitability score
5. Generate human-readable explanation

The system is designed for:

- extensibility
- scientific interpretability
- performance scaling

---

## 📊 Visualization Model

The parameter space is rendered as a continuous field:

- X-axis → α (fine-structure constant)
- Y-axis → strong force strength
- Color → habitability score

This allows identification of:

- viable regions
- unstable zones
- transitional boundaries
